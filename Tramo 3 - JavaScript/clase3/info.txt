Resumen de pasos resumidos:
Creamos HTML:
* Creamos un formulario
* Dentro del formulario ubicamos los campos que completa el usuario y cada uno con una id
* Debemos incluir un botón que tenga el llamada a la función onclick
* Incorporamos una elemento (div o p) para la respuesta. Ese elemento debe tener una id.
* Nos aseguramos que el script esté vinculado con el HTML con la etiqueta script

Creamos el JavaScript
* Creamos la funció que es llamada desde el botón, por el usuario
* Dentro de la función:
 - Obtenemos los valores de los campos y los guardamos en variables
 - Escribimos los pasos de la consigna
 - Escribimos el mensaje del resultado o del error
